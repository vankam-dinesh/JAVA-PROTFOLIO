Latest Projects

Food Recipe Guide Application


The Food Recipe Guide Application is a web-based project developed using HTML, CSS, and JavaScript. The application allows users to explore a wide variety of recipes with detailed information such as ingredients and preparation steps. It integrates a public Recipe API to fetch real-time recipe data, enabling dynamic content updates. Features like recipe search, category-based browsing, and interactive UI elements were implemented to enhance usability. The project focuses on a responsive design, clean user interface, and smooth user experience, making it easy for users to discover and learn new recipes efficiently.

Hospital Management System


The Hospital Management System is a console-based application developed using Java and MySQL to manage hospital operations efficiently. The system allows users to add and view patient details, manage doctor information, and book appointments while ensuring data consistency. It uses JDBC for database connectivity and performs CRUD operations through secure Prepared Statements. A well-structured relational database design with foreign key constraints is implemented to maintain data integrity. The project emphasizes modular programming, exception handling, and clean backend logic, serving as a strong foundation for understanding Java–MySQL integration and database-driven application development.

Banking System


The Banking System is a console-based application developed using Java and MySQL to perform essential banking operations efficiently. The system supports functionalities such as account creation, balance inquiry, deposits, withdrawals, and transaction tracking. It uses JDBC for secure database connectivity and implements CRUD operations through Prepared Statements to prevent SQL injection. A relational database design is used to ensure data consistency and accuracy of transactions. The project focuses on object-oriented programming principles, exception handling, and modular code structure, providing a strong understanding of banking workflows and Java–database integration.
